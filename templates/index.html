GET /api/current Active Stories

Fetch all currently active (non-expired) stories from a Telegram user or channel.

GET /api/special Download Story

Download a specific story by providing username and story ID. Returns a temporary download URL valid for 60 minutes.

Query Parameters

Parameter Type Required Description
username string Yes Telegram username without @ symbol
storyid integer Yes Unique story identifier (numeric)

Example Request

cURL 
curl -X GET "/api/special?username=durov&storyid=1234"

Response Format

200 OK 
{
  "success": true,
  "username": "durov",
  "story_id": 1234,
  "type": "Active",
  "media_type": "video",
  "date": "2026-01-09 14:30:00",

  "expires_in": "60 minutes",

GET /api/direct Direct Link

Download story directly using a Telegram story URL. Automatically parses username and story ID from the URL.

Query Parameters

Parameter Type Required Description
url string Yes Full Telegram story URL (URL-encoded)

Supported URL Formats

https://t.me/username/s/storyid Standard story link
https://telegram.me/username/s/storyid Alternative domain
https://t.me/c/channelid/storyid Private channel story

Example Request

cURL 
curl -X GET "/api/direct?url=https://t.me/durov/s/1234"

Response Format

200 OK 
{
  "success": true,
  "username": "durov",
  "story_id": 1234,
  "type": "Pinned",
  "media_type": "photo",
  "date": "2026-01-09 14:30:00",
  "timestamp": 1736432200,
  "caption": "Amazing photo!",
  "download_url": "https://tmpfiles.org/dl/789012/story.jpg",
  "expires_in": "60 minutes",
  "source_url": "https://t.me/durov/s/1234",
  "api_dev": "@ISmartCoder",
  "api_channel": "@abirxdhackz"
}
400 Bad Request 
{
  "success": false,
  "error": "Invalid Telegram story URL format. Expected: https://t.me/username/s/storyid",
  "api_dev": "@ISmartCoder",
  "api_channel": "@abirxdhackz"
}

Response Fields Reference

Common Fields

success Boolean indicating request status
error Error message (only on failure)
api_dev API developer contact
api_channel Official Telegram channel

Story Object

story_id Unique story identifier
type Active | Pinned | Archived
media_type photo | video | document
caption Story caption text (may be empty)

Download Fields

download_url Temporary file download link
expires_in Link expiration time (60 min)
date Human-readable date string
timestamp Unix timestamp (seconds)

Rate Limiting

API requests are subject to Telegram's rate limits. Excessive requests may result in temporary blocks. Recommended: max 30 requests/minute.

Download Links

Download URLs are temporary and expire after 60 minutes. Store content locally if needed for longer periods.

Authentication

No API key required. All endpoints are publicly accessible. For private channels, ensure the bot has access.
